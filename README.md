🎯 Daily Habit Tracker System (CLI)

Sistem Pelacak Kebiasaan Harian berbasis Command Line Interface (CLI) yang dibangun menggunakan bahasa pemrograman Python. Aplikasi ini dirancang untuk membantu Anda melacak, konsisten, dan membangun kebiasaan positif setiap harinya dengan sistem pencatatan otomatis (streak system).

✨ Fitur Utama

Tambah Kebiasaan Baru: Daftarkan target aktivitas harian Anda beserta deskripsinya secara fleksibel.

Pencatatan Kemajuan Real-time: Catat penyelesaian tugas harian secara interaktif.

Sistem Streak Dinamis (🔥): Menghitung rutinitas harian berturut-turut tanpa jeda dan menyimpan rekor pencapaian tertinggi Anda.

Statistik & Laporan Lengkap: Pantau seberapa sering Anda konsisten melakukan aktivitas positif Anda.

Penyimpanan Lokal Otomatis (JSON): Data Anda aman tersimpan dalam file lokal habit_data.json dan otomatis diperbarui setiap kali ada perubahan.

Desain UI Terminal yang Bersih: Menu interaktif yang ramah pengguna.

🛠️ Teknologi Yang Digunakan

Python 3.x

JSON (JavaScript Object Notation) untuk sistem database lokal.

OS & Datetime standard libraries (Tanpa perlu menginstal dependensi pihak ketiga).

🚀 Cara Menjalankan Project

Ikuti langkah-langkah di bawah ini untuk menjalankan program di komputer Anda:

Clone Repositori ini:

git clone [https://github.com/USERNAME-ANDA/daily-habit-tracker.git](https://github.com/USERNAME-ANDA/daily-habit-tracker.git)
cd daily-habit-tracker


Jalankan Aplikasi:
Pastikan Anda sudah menginstal Python di komputer Anda, lalu jalankan perintah:

python habit_tracker.py
